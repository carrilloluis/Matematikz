\subsubsection*{Proporciones}
Hallar la variable (término faltante) en la proporción  :
\begin{itemize}
\item{$50 - 42 = 25 - x$}
\item{$45{.}3 - x = 18{.}0 - 0{.}03$}
\item{$\dfrac1{3} - \dfrac1{7} = \dfrac{5}{6} - x$}
\item{$\dfrac{x}{\Big(\dfrac{1}{5}\Big)} = \dfrac{6}{2}$}
\item{$\dfrac{\Big(\dfrac{1}{4}\Big)}{x} = \dfrac{x}{\Big(\dfrac{9}{16}\Big)}$}
\item{$\dfrac{8}{x} = \dfrac{16}{4}$}
\end{itemize}
